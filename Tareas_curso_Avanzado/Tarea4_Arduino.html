Proyecto

He implementado una especia de vumetro con led, en Arduino Mega, (en lo siguiente lo detallo), y me parece interesante hacer la adaptación a la placa LilyPad, para poder incorporarlo a alguna prenda de vestir y hacerla así mucho más vistosa.

Un ejemplo de prenda que puede incorpora el diseño puede ser una corbata, o un sombrero.

Diseño en Arduino Mega

El tema es sencillo, consiste en a partir de la señal que capte un microfono, encender y apagar una serie de led.

El problema es que un microfono ("sin aplificar"), genera una señal con valores demasiado pequeños para que las entrada analógica de Arduino puedan percibir los cambios.

Por tanto debemos implementar un circuito amplificador, que nos ajuste la señal a un rango de voltages de 0-5 V.

Esquemático.

Referencia del circuito: Circuito Amplificador Arduino

Montaje.

Codigo

int sensorPin = A0; //Pin del micro 
int sensorValue = 0;  //Valor leido por micro

int led=2;
int led1=4;
int led2=6;
int led3=8;
int analogPin=0;

void setup() {

  pinMode(led, OUTPUT); 
  pinMode(led1, OUTPUT);
  pinMode(led2, OUTPUT);
  pinMode(led3, OUTPUT);
  Serial.begin(9600);
}

void loop() {

  int mn = 1024;     // mn only decreases
  int mx = 0;        // mx only increases

  // Perform 10000 reads. Update mn and mx for each one.
  for (int i = 0; i < 1000; ++i) {
    int val = analogRead(analogPin);
    mn = min(mn, val);
    mx = max(mx, val);
  }

  Serial.print("m=");
  Serial.print(mn);  
  Serial.print(" M=");
  Serial.print(mx);  
  Serial.print(" D=");
  Serial.print(mx-mn);  
  Serial.println();  

  int dif=mx-mn;

  if (dif<30){   //Establecemos el umbral a 256 (entre 0-1024)

    digitalWrite(led, LOW);
    digitalWrite(led1, LOW);  
    digitalWrite(led2, LOW);  
    digitalWrite(led3, LOW);    
    Serial.println("Un led encencido");    
  }

  else if ((dif>=30) &&(dif<200) ) {  
    digitalWrite(led, HIGH);
    digitalWrite(led1,LOW);  
    digitalWrite(led2, LOW);  
    digitalWrite(led3, LOW);    
    Serial.println("Dos leds encendido");

  }


  else if ((dif>=200) &&(dif<330) ) {  
    digitalWrite(led, HIGH);
    digitalWrite(led1,HIGH);  
    digitalWrite(led2, LOW);  
    digitalWrite(led3, LOW);    
    Serial.println("Dos leds encendido");

  }

  else if ((dif>=330) &&(dif<470) ) {  
    digitalWrite(led, HIGH);
    digitalWrite(led1, HIGH);  
    digitalWrite(led2, HIGH);  
    digitalWrite(led3, LOW);    
    Serial.println("Tres leds encendido");

  }

else {

    digitalWrite(led, HIGH);
    digitalWrite(led1, HIGH);  
    digitalWrite(led2, HIGH);  
    digitalWrite(led3, HIGH);    
    Serial.println("Todos leds encendido");
}

  sensorValue = analogRead(sensorPin);    
  Serial.println(sensorValue); 
}

Diseño en Arduino LilyPad

Para realizar las comunicaciones LilyPad da la posibilidad de utilizar, una especie de hilos ("como los que se utilizan los sastres") conductores.

LilyPad también ha diseñado una especie de LED planos, que ya incorporan resistencia interna, especialmente destinados a ser aplicados a prendas de vestir.

Los led los podemos conectar respetando el mismo número de puerto que usábamos en Arduino Mega, y así evitar modificar esto en el código.

Podemos verlo en la siguiente imgagen:

Respecto a la entrada igualmente podemos utilizar el puerto A0, siendo por tanto el código totalmente compatible.

Si es interesante realizar un montaje del circuito amplificador lo más compacto posible, y buscar alguna batería lo menos aparatosa posible.
